### Problem 1.
Debit Card Number

Write a program that receives 4 integers with up to 4 digits each and prints them in 4-digit debit card format.

Examples:

Input:
12
433
1
5331

Output:
0012 0433 0001 5331

Input:
9182
4221
53
125

Output:
9182 4221 0053 0125
